OWNER PASSWORD OPTIONS

1. If Owner can still log in:
   - Click Change Password in the sidebar footer.
   - Enter current password and new password.

2. If Owner forgot password in local XAMPP:
   - Open phpMyAdmin.
   - Select/import: database/03_OWNER_FORGOT_PASSWORD_LOCAL_XAMPP.sql
   - Login using temporary password: Owner@12345
   - The system will force the Owner to create a new password.

This is free and does not require email, SMTP, or hosting.
Do not use email verification/password reset unless you have a configured email sender.
